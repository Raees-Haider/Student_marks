# Student Marks Management System
A simple command-line based Student Marks Management System built in Ruby. This project allows users to manage school sections, add students, and view academic performance such as position holders.

# How to Run

## Make sure you have Ruby installed

```bash
ruby -v
```

## Clone the repository

```bash
git clone git@github.com:Raees-Haider/Student_marks.git
```

```bash
cd Student_marks
```

## Run the main file:

```bash
ruby main.rb
```

## Menu Options

When you run the program, you will see:
```bash
===============================
	       Marks System
===============================
1.Display Sections
2.Display students in a section
3.Add a section
4.Add a student
5.Position holder in a Section
6.Exit
Choose an option:
```
